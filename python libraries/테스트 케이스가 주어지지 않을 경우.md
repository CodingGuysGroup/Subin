
# 문제에 테스트 케이스가 주어지지 않았을 경우

<br/>

EX1) https://www.acmicpc.net/problem/10951

``` python
# EX1
import sys
input = sys.stdin.readline

try:
    while True:
        a, b = map(int, input().split())
        print(a+b)
except:
    exit()

```
